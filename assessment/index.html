HTML

  • Staat voor hypertext markup language.
  • Is de standaard markup language voor het maken van webpaginas
  • Beschrijft de structuur van de webpagina
  • Bestaat uit een serie elementen die de browser vertelt hoe de inhoud weer te geven
  • -Elementen geven inhoud een label, bv: dit is een paragraaf met tekst, dit is een linkje, etc

CSS

Staat voor Cascading Style Sheets
Een stylesheet geeft opmaak en style aan de content

Opbouw met benamingen:
simple_selector.class_selector combinator+ element.selector:pseudo-class{
      property:value;
      property:value;
      property:value;
}

CSS kan internal of external zijn
De selector bepaalt de opmaak van de subject in het html document

Selector

id selector(uniek id per tag) of class selector (meerdere tags met dezelfde class).
#header geeft eigenschappen aan de html tag met het attribuut "id" met de waarde "header"
.header geeft eigenschappen aan de html tags met het attribuut "class" met de waarde "header".
